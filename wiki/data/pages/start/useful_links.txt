====== Useful links ======

  * LArSoft code repository: [[https://github.com/LArSoft|LArSoft github repository]]
  * Tutorial on how to install LArSoft (Fermilab account required)[[https://cdcvs.fnal.gov/redmine/projects/sbndcode/wiki/How_to_setup_your_directory_and_launch_your_first_job|Redmine, How to setup your directory and launch your first job]]
